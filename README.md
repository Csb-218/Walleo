
# Z wallet

A decentalized crypto wallet.


## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/Csb-218/Z-wallet.git

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
